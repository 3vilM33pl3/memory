# `memory stats`

`memory stats` prints a quick memory and project summary.

Use it for lightweight inspection when you want counts, not a full health diagnosis.

## Common Usage

```bash
memory stats
memory health
memory doctor
```

## When To Use It

- Use `memory stats` for a compact database/project summary.
- Use `memory health` for backend reachability and service identity.
- Use `memory doctor` for configuration, database, watcher, embedding, skill, and environment diagnostics.
