# Agent project-init prompt

Use this after Memory Layer is installed and you want an agent to configure one repository.

```text
You are configuring this repository for Memory Layer. Do not install the package. Inspect existing `.mem` and `.agents` files, run `memory wizard --dry-run`, explain planned writes, then run `memory wizard` only after the preview is safe. Verify with `memory status --project <project-slug>` and a simple query.
```

## Verify

The agent should report changed files, project slug, health/status output, and any follow-up checks.
