# Agent install prompt

Use this when you want an agent to install Memory Layer for you.

```text
You are installing Memory Layer for me. Work in the terminal, explain before using sudo, and stop before destructive changes.

Find the correct install path for this OS, configure PostgreSQL with pgvector, run `memory wizard --global`, run health checks, and report where config files live. Do not delete data or overwrite existing config without asking first.
```

## Verify

The agent should finish with `memory doctor`, `memory health`, installed version, service status, and next commands.
