# Showcase

Memory Layer is strongest when the user can see evidence, not just output text.

![Memory Layer cited query answer](/images/tui/query-tab.png)

## What to look for

- Ranked memories behind a query answer.
- Citations, source paths, and graph retrieval diagnostics.
- Live agent and watcher status.
- Activity history that can become future memory.
- Review queues for proposed memory replacements.
