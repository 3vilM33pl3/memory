# Agent briefings

Briefings help a new or returning agent understand recent project state before spending tokens rediscovering context.

```bash
memory resume --project <project-slug>
memory up-to-speed --project <project-slug>
memory activities --project <project-slug> --limit 20
```

Use briefings at task start, after interruptions, and before continuing work that another agent began.

## Verify

A useful briefing includes recent work, warnings, next actions, and changed memories or activities.
