# Support bundle

Memory Layer does not need a support bundle command for this page to be useful. Until a dedicated command exists, collect the same information manually.

## Collect

```bash
memory --version
memory doctor
memory health
memory status --project <project-slug>
memory mcp status --project <project-slug>
memory watcher manager status
```

## Redact

Remove database URLs, API keys, tokens, private project names if needed, captured prompts, and file paths that reveal confidential data.

## Include

- OS and install method.
- Project slug.
- Whether this is packaged or dev-profile service.
- Recent relevant logs.
- Steps to reproduce.
