# Generic watcher

Project watchers observe repository changes, debounce bursts, and send useful activity to the service.

```bash
memory watcher run --project <project-slug>
memory watcher status --project <project-slug>
```

Foreground and legacy service-managed watchers are advanced flows. Use them when the watcher manager does not match the agent environment.

## Verify

```bash
memory status --project <project-slug>
memory activities --project <project-slug> --kind watcher --limit 10
```
