# Codex profile

The recommended Codex path is the watcher manager. It detects live Codex sessions, maps them to projects, and starts a watcher for the session.

```bash
memory watcher manager enable
memory watcher manager status
```

## Verify

Open the TUI Agents and Watchers tabs, or run:

```bash
memory status --project <project-slug>
```
