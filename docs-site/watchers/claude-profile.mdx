# Claude profile

Memory Layer has watcher concepts for Claude-linked work, but exact client session details can vary by local setup.

Use the generic watcher path unless your local Claude setup has a maintained profile. Keep the project explicit, verify the watcher from `memory status`, and prefer activity evidence over assuming a session was captured.

## Verify

Use `memory status --project <project-slug>` and the TUI Watchers tab to confirm any watcher attached to the intended project.
