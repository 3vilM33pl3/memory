# CLI reference

Use command-specific help as the source of truth:

```bash
memory --help
memory <command> --help
```

| Command family | Purpose |
|---|---|
| `memory wizard` | Configure global or project setup. |
| `memory doctor` | Diagnose configuration and environment. |
| `memory health` | Check backend service health. |
| `memory service` | Run and manage the service. |
| `memory status` | Show aggregate diagnostics. |
| `memory query` | Ask evidence-backed project questions. |
| `memory resume` / `memory up-to-speed` | Generate briefings. |
| `memory remember` / `memory capture` / `memory curate` | Capture and curate memory. |
| `memory watcher` | Manage watcher processes. |
| `memory mcp` | Run and inspect MCP. |
| `memory eval` | Run evaluation suites. |

## Verify

Run `memory <command> --help` before copying options into automation.
